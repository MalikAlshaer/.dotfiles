#### Requirements:
##### alacritty:
- ``picom`` (unless you already have a compositor)

##### neovim:
- ``neovim`` (10 or newer)\
- [``vim-plug``](https://github.com/junegunn/vim-plug) (plugin manager)\
- [nerdfont of your choice](https://www.nerdfonts.com/font-downloads)

##### i3:
- ``i3-wm`` (obviously)\
- ``nm-applet`` (network manager)\
- ``feh`` (for wallpapers)\
- ``picom`` (compositor)\
- ``rofi`` (app launcher)
